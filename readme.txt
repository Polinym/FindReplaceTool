All this does is the standard find/replace, but in all files
within all subdirectories of the folder you choose.

How you use this functionality, good or bad, rests with you.

-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-

H O W   T O   U S E 

1. Press [Select Directory] to select a folder.
Note! All subdirectories of the folder will be included automatically!

2. Enter the word you want to Find in [Find:]

3. Enter the word you want to replace it with in [Replace:]

4. Press [Replace All In Directory]. It will do just that.

5. Profit
The number of occurances replaced in each file will appear below
in the output text box.
